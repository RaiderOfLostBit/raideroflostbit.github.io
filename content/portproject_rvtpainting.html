Based on an idea for an RTS game by two friends and me, I needed to develop a toolset for Unreal Engine that would allow us to texture large landscapes from close-up views to large top-down vistas as easily, efficiently and non-destructively as possible.

This resulted in the RVT Painting Plugin, which consists of three main components:

  • RVT Landscape Auto Material
  • RVT Painting Editor Mode
  • RVT Landscape Decals

Initially developed for Unreal Engine 4, the system was later ported to Unreal Engine 5.

Runtime Virtual Texturing (RVT)

The toolset is based on Unreal's Runtime Virtual Texturing System, which is basically a caching method for materials to reduce render times while increasing VRAM usage. As this system is specifically designed for the optimization of landscapes, it provides the perfect foundation for our needs.

RVT Landscape Auto Material

The basis of the custom tools is a special landscape material, the RVT Landscape Auto Material. It automatically places materials on the landscape by using landscape information such as the slope angle and height.

Normally in Unreal, you have to create a single complex landscape material with all the layers you need already defined, which can then be painted or placed onto the landscape. If you have a limited team size, it can be difficult to maintain such a material.

To solve this problem, the custom material uses Unreal's Material Layer System, which works similar to layers in common image editing programs. This makes it possible to move the shader code of the layers as well as their blending functions into their own assets, which leads to a significant reduction in the complexity of the actual landscape material.

In addition, packed masking information is stored in the RVT system for procedural placement of vegetation and physical materials. This enables the individual layers, RVT Painting and RVT Decals, to define their own grass or physical types usage.

RVT Painting

The core of the toolset is the RVT Painting Editor Mode, which is used for manual texturing of landscapes and offers a more modular painting approach than the built-in tools.

It uses the RVT system's ability to render meshes with their materials into its cache and then project them onto a landscape. On this basis, a grid-based painting system is created allowing the dynamic addition or removal of paint layers without prior implementation in the landscape material.

Each paint layer has its own RGB weightmap, so three different ground types can be painted per layer. Ground types are defined within an assignable Material Instance also using Unreal's Material Layer system. In this way, ground types can be shared with the landscape material. Available blending functions influence the shape of the paint brush. In addition to the Material Instance, a sort index determines the order of the layers among each other.

The main advantages of the systems are:

  • Dynamic creation or removal of painting layers.
  • Dynamic sorting of painting layers as well as locking layers and switching visibility.
  • Easy swapping of the used Material Instance for a layer, changing its ground types and blending.
  • Custom controls for painting, taking into account industry shortcuts.
  • Each ground type can have its own physical material as well as its own grass type.

Caveats:

  • Unreal's World Composition or World Partition systems are not supported.
  • By splitting the system into different assets with different functionality, it is initially more difficult for designers to understand the workflow.
RVT Landscape Decals

RVT Decals offer a performant alternative to regular decals and are used to add rich detail to the landscape. The materials of these decals can be setup similar to regular decals or use the same layered approach as the Landscape Auto Material or RVT Painting. In combination with Blueprint Brushes, particularly convincing results are achieved.

Since the RVT system allows changes in runtime, RVT Decals can also be used in conjunction with gameplay elements and allow a high number of instances to be added without major performance losses.
